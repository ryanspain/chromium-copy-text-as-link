Choose your preferred copy command

This changes what copy command is used when you highlight text and use the right-click context menu option from a web page.

Copy text as page link

Create a link that points directly to the current page, perfect for sharing entire articles or documents.

Copy text as fragment link

Create a link that highlights specific text on the page, ideal for referencing specific passages or quotes.

Advanced settings

This are additional settings for the extension that enable fine-tuning of the copy text as link behavior. Check the description for each setting before making changes.

Link format
Controls the format of the link that's copied to the clipboard. HTML is the default and useful for pasting links into rich-text editors like Outlook, while Markdown is great for platforms like GitHub.

HTML example:
<a href="https://example.com">Link text</a>
Markdown example:
[Link text](https://example.com)
